Overview

Class: icinga

This is the icinga module. This module install all things required to setup Icinga 2.

Parameters

Here is the list of parameters used by this module.

  • `type` Specify if client or server components should be installed Default value is client

  • `plugins` Specify one or more plugins that should be installed Default value is none (not implemented yet)

  • `exported_ressources` Specify if exported resources should be used Default value is false (not implemented yet)

  • `server_acl` Specify the Icinga servers that are allowed to access monitoring client Default value is 127.0.0.1

Variables

No additonal variables are required for this module
